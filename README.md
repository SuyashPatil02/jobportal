# Job Portal Project

This is a Django-based job portal web application.

## Features
- User registration & login
- Job listing
- Apply for jobs
- Employer dashboard

## Tech Stack
- Python
- Django
- HTML, CSS

## How to run
1. Install dependencies
2. Run migrations
3. Start server using:
