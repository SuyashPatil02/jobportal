# 💼 Job Portal Project

This is a **Django-based Job Portal web application** where users can search and apply for jobs, and HR can manage applications through an employer dashboard.

---

## 🧠 System Flow

- **Admin** adds jobs  
- **User** logs in → searches jobs → applies  
- Application is saved as **Pending**  
- **HR** reviews applications in employer dashboard  
- HR can **Approve / Reject**  
- User dashboard updates the **status**

---

## ⚙️ Features

- 🔐 User Registration & Login  
- 🔎 Job Search & Filtering  
- 📄 Job Application with Resume Upload  
- 📊 User Dashboard (Track Applications)  
- 🧑‍💼 Employer Dashboard (HR Panel)  
- ✔️ Approve / ❌ Reject Applications  
- 🟡 Status Tracking (Pending / Shortlisted / Rejected)  
- 🎨 Modern UI with job cards and status badges  

---

## 🛠️ Tech Stack

- Python 🐍  
- Django 🌐  
- HTML, CSS 🎨  
- SQLite Database 🗄️
